# Skill: landing-reviewer

## Purpose
Validate the landing page implementation and report deltas and exceptions.

## Commands
- `yarn lint`
- `yarn build`
- `yarn test:e2e`

## Output
A checklist that includes:
- which DS components were used
- bespoke exceptions (why)
- failures + suggested fixes
